The following exception numbers are used by durexForth.

-1 :: ((abort))
-2 :: ((abort"))
-4 :: stack underflow
-8 :: dictionary overflow (out of memory)
-10 :: division by zero
-13 :: undefined word
-16 :: attempt to use an empty string as a name
-28 :: user interrupt (triggered by kbd:[RESTORE] or `brk` instruction)
-37 :: file I/O exception
